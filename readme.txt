== Description ==
Custom payment gateway for WooCommerce to integrate XpayCheckout.
This plugin is free to use, its just a little contibution from my side. if you want to hire me for other task or extra modification on this plugin, like subuscription or direct payment link then hire me via freelancer https://freelancer.com/u/fantasyinfo

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/woocommerce-xpaycheckout` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to WooCommerce -> Settings -> Payments and configure the XpayCheckout settings.
